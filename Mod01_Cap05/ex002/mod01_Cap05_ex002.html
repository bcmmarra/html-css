Para representar uma tag escrita na tela igual feito acima é necessário usar caractéres especiais como os usados acima, & lt e & gt.

Olhe no código fonte os caractéres utilizados para representar os símbolos a seguir: ®, ©,  , ™, €, £, ¥, ¢, ∅, ∑, Δ, ←, ↑, →, ↓. Entre outros. LISTA COMPLETA no site da W3schools.

Inserindo Emojis

Para acrescentar Emojis na página é necessário inserir o código Unicode no site Emojipedia

Após escolher uma categoria de emoji, encontre o Codepoints, U+1F604 (por exemplo), para inserir o emoji no site use o código adaptado &#x e adicione o código obtido no Codepoints sem as letras U+ ficando com resultado assim: 😄.
