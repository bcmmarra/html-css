FORMATAÇÃO DE TEXTOS

A grande diferença entre o HTML4 e HTML5 é a semântica (Sentido interno das palavras).

Exemplo de texto sem semântica

Juvenal era um sujeito de muito sorte. E já começou de pequeno, onde morou na Rua Marques de Lira Filho, um local de fácil acesso ao Centro da cidade.

Exemplo de texto com semântica

Juvenal era um sujeito de muito sorte. E já começou de pequeno, onde morou na

Rua Marques de Lira Filho
, um local de fácil acesso ao Centro da cidade.

Agora usamos a tag <address> para dar significado ao destaque que fizemos no endereço do Juvenal, em alguns casos o navegador de celular pode até sugerir que você veja o mapa do local e trace a rota para chegar lá.

Tags de Formatação de Texto

Negrito e Itálico

Exemplo de texto em Negrito com a tag <strong>, possui semântica indicando que o texto possui força;

Exemplo de texto em Itálico com a tag <em>, possui semântica indicando que o texto possui ênfase;

MARCA TEXTO

Exemplo de texto em Marcado tipo marca texto com a tag <mark>, possui semântica indicando que o texto possui ênfase descomunal;

TEXTO MENOR

Exemplo de texto em "letras miudas" com a tag <small>, possui semântica indicando que o texto é texto de letras pequenas, ex. letras miudas de contratos;

Texto DELETADO

Exemplo de texto deletado com a tag <del>, possui semântica indicando que o texto esta ali é para ser visto pelo usuário mas desconsiderado, muito usado em promoções tipo de tanto por tanto;

Texto INSERIDO

Exemplo de texto em Inserido com a tag <ins>, possui semântica indicando que o texto está ali e deve ser lido e você deve presta atenção nele, pois possui grande importante, substituido a tag <u>;

H2O e 102

Exemplo de texto em H2O e 102 com as tags <sub> e <sup>, possui semântica indicando que o texto representa um formula ou equação, letra pequena acima do texto ou abaixo do texto;

Exemplo de código em Python

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é ímpar') print('FIM DO PROGRAMA')

Fazendo o código assim o HTML vai colocar cada linha de código na frente uma da outra. Para resolver esse problema basta colocar a tag <code> entre uma tag <pre>


        num = int(input('Digite um número'))
        if num % 2 == 0:
            print(f'O número {num} é par')
        else:
            print(f'O número {num} é ímpar')
        print('FIM DO PROGRAMA')

A tag <pre> mantem exatamente os espaços que colocamos no código no HTML;

Essa tag BDO passei o mouse sobre o sigla para ver o significado, ela coloca o texto digitado invertido, ele possui 2 configurações no parametro dir "direção do texto" rtl ou ltr.

rtl = da direita para a esquerda (right-to-left)
ltr = da esquerda para a direita (left-to-right).
