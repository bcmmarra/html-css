Imagens Pequenas como Plano de Fundo

background-repeat: repeat; ou deixa sem a declaração - Configuração Padrão, a imagem vai ser repetir até cobrir todo o espaço disponivel, mesmo que precise cortar parte da imagem.

background-repeat: repeat-x; - Configuração para repetir apenas no eixo x do espaço "horizontal".

background-repeat: repeat-y; - Configuração para repetir apenas no eixo y do espaço "Vertical".

background-repeat: no-repeat; - Configuração para não repetir a imagem, a imagem vai mostrar no seu tamanho Padrão apenas 1 vez.

background-position: left top; - Configuração para aplicar a imagem no elemento, por padrão ele apresenta a imagem left top, ou seja, começa alinhado a esquerda na parte de cima da imagem.

Imagens Grande como Plano de Fundo

Outra coisa que podemos fazer é redimensionar a imagem para forçá-la a caber na caixa. Por padrão, nenhum redimensionamento será aplicado, e a imagem será exibida do seu tamanho natural. Porém, podemos usar a propriedade backgroundsize para alterar esse comportamento.

background-size: 100px; - Colocando essa opção ele vai reduzir a imagem para o comprimento definido. até completar todo o espaço do elemento.

background-size: 100%; - Colocando essa opção ele vai reduzir a imagem para o comprimento 100% do espaço da caixa até completar todo o espaço do elemento e se a altura for maior ele vai repetir a menos que coloquemos a opção no-repet e neste caso o espaço vai ficar sem a imagem de fundo.

background-size: cover; - Colocando essa opção ele muda o tamanho da imagem para que ela seja sempre totalmente exibida na tela, sem nenhum corte.

background-size: contain; - Colocando essa opção ele redimensiona a imagem para que ela cubra o contêiner, mesmo que para isso ocorram alguns eventuais cortes.

Imagens Grande como Plano de Fundo

A última propriedade que podemos configurar é o vínculo (attachment) da imagem de fundo com o resto do documento, principalmente se o conteúdo for maior do que a altura da página e seja necessário vazar uma rolagem vertical.

background-attachment: scroll; Padrão a imagem de fundo vai rolar junto com o conteúdo.

background-attachment: fixed; A imagem de fundo vai ficar fixada enquanto o conteúdo vai sendo rolado.

Simplificando as coisas

Assim como já vimos várias vezes em nosso material, existe também a possibilidade de usar uma shorthand para simplificar o uso de propriedades que se apliquem ao fundo de uma caixa. A propriedade abreviada background pode ser declarada agrupando as seguintes configurações:

  • background-color
  • background-image
  • background-position
  • background-repeat
  • background-attachment

Sendo assim, no lugar use:

  • background: black url('imagens/wallpaper002.jpg') center center no-repeat fixed
