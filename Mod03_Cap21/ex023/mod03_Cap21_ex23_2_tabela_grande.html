HIERARQUIA DE TABELAS

  • TABLE = Area da Tabela
    • CAPTION = Titulo da tabela
    • THEAD = Area do cabeçalho da Tabela
      • TABLE ROW <tr> </tr> = Linha da tabela
        • TABLE HEADER <th> </th> = Cabeçalho da tabela
      • TABLE ROW <tr> </tr> = Linha da tabela
        • TABLE DATA <td> </td> = Dado de tabela
    • TBODY = Area do corpo da Tabela
      • TABLE ROW <tr> </tr> = Linha da tabela
        • TABLE HEADER <th> </th> = Cabeçalho da tabela
      • TABLE ROW <tr> </tr> = Linha da tabela
        • TABLE DATA <td> </td> = Dado de tabela
    • TFOOT = Area do rodepé da Tabela
      • TABLE ROW <tr> </tr> = Linha da tabela
        • TABLE HEADER <th> </th> = Cabeçalho da tabela
      • TABLE ROW <tr> </tr> = Linha da tabela
        • TABLE DATA <td> </td> = Dado de tabela
