Desafios HTML5 - CSS3 - JS

Durante as aulas de HTML5 e CSS3, tivemos vários desafios para pratirar, focando o seu conteúdo em tudo que aprendemos até aqui... Toda a base começa sempre com essas tecnologias.

DESAFIOS

"Clique no abaixo para entrar nos Desafios."

    M01C05: CARACTERES, PARÁGRAFOS E QUEBRAS DE LINHA

  • Desafio 001 - Desafio das Mensagens
    •

    Exibir um texto com uma mistura de paragrafos e quebra de linhas, sem o uso de varios <br>.

    M01C06 IMAGENS E FAVICONS

  • Desafio 002 - Desafio das Imagens
    •

    Criar uma página que tenha duas imagens: uma no conteúdo da página e um ícone de favoritos.

  • Desafio 003 - Desafio do Mapa Mundi
    •

    É um mapa montado por nove imagens diferentes. É uma ótima maneira de treinar a diferença entre parágrafos e quebras de linha, além de saber inserir imagens.

  • Desafio 004 - Desafio dos Meus Emojis favoritos
    •

    Vamos juntar agora os conceitos de listas e emojis

    M01C10 LIGAÇÕES EM TODA PARTE

  • Desafio 005 - Desafio Meu perfil social
    •

    Vamos agora criar uma página que divulgue suas redes sociais. Teremos uma foto, uma breve descrição e os links para os seus perfis.

  • Desafio 006 - Desafio das Tags
    •

    O desafio aqui é arrumar uma maneira que, ao passar o mouse sobre um termo, mostre qual é a tag utilizada para fazer isso

    M01C11 IMAGENS DINÂMICAS, ÁUDIO E VÍDEO

  • Desafio 007 - Desafio da Imagem flexível
    •

    Quando a janela está bem pequena, mostre uma imagem mais estreita, que não ocupe tanto espaço. Porém, quando o usuário aumentar a largura do navegador, você pode substituir a imagem menos ampla por uma maior.

  • Desafio 008 - Desafio Criando uma Navegação
    •

    O desafio dessa vez é criar navegação entre páginas. Você deve criar uma página simples, com link para mudar de uma página para outra e conseguir retornar para a página principal

  • Desafio 009 - Desafio Seus vídeos favoritos
    •

    O desafio aqui é criar um site onde você possa indicar quatro vídeos legais para seu visitante poder visitar.

    M02C16 O MODELO DE CAIXAS

  • Desafio 010 - Desafio Site do Android
    •

    Vamos criar uma página com tudo aquilo que foi ensinado até aqui.

    M02C19 IMAGENS DE FUNDO

  • Desafio 011 - Desafio do Astronauta
    •

    Desafio é unir duas imagens

    M02C20 MINI-PROJETO CORDEL

  • Desafio 012 - Desafio do Cordel
    •

    Nesse desafio, vamos transformar uma poesia em um site, incluindo efeitos de parallax nas imagens aplicadas ao fundo da página.

    M02C21 TABELAS

  • Desafio 013 - Desafio das Tabelas
    •

    Nesse desafio, vamos trabalhar com mesclagem de linhas e colunas de tabelas com rowspan e colspan.
